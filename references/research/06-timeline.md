# Tim Cook — 人物时间线

## 关键里程碑

| 时间 | 事件 | 意义 |
|------|------|------|
| 1960年11月1日 | 出生于Alabama州Robertsdale | 南方小镇成长背景，塑造了Southern politeness和对少数群体的同理心 |
| 1978年 | 进入Auburn University（工业工程专业） | 工程思维和系统优化的起点 |
| 1982年 | Auburn大学毕业（工业工程学士） | — |
| 1982-1994年 | IBM工作12年，供应链和制造领域 | 运营管理能力的核心培养期，学会了大规模系统优化 |
| 1988年 | Duke University Fuqua商学院MBA毕业 | 补充商业战略视角 |
| 1994年 | 加入Intelligent Electronics | 短暂过渡 |
| 1995-1998年 | Compaq副总裁（材料采购） | PC行业运营经验 |
| **1998年3月** | **加入Apple**，任全球运营高级副总裁 | **人生转折点**。Cook放弃Compaq的"安全"工作，被Jobs说服加入当时濒临破产的Apple。"Five minutes into my initial interview with Steve, I just wanted to throw caution and logic to the wind." |
| 1998-2000年 | 重建Apple供应链 | 库存周转天数从30天→6天→约5天。关闭全球大部分仓库，推行JIT制造 |
| 2002年 | 升任全球销售和运营执行副总裁 | 权力扩大 |
| 2005年 | 升任首席运营官（COO） | 成为Apple二号人物 |
| 2004年 | Jobs第一次病假（胰腺癌手术），Cook代理管理 | 首次证明能在Jobs缺席时稳定公司 |
| 2009年1-6月 | Jobs第二次病假（肝脏移植），Cook代理CEO | 公司运营平稳，市场信心不减。Cook曾提出捐肝被Jobs拒绝 |
| **2011年8月24日** | **正式成为Apple CEO** | Jobs辞去CEO，董事会任命Cook。Jobs嘱咐："Just do what's right." |
| 2011年10月5日 | Steve Jobs去世 | Cook独自掌舵的真正开始 |
| 2012年9月 | Apple Maps危机 | Cook发布罕见的公开道歉信，推荐用户使用竞品地图。这是Cook展示务实（vs Jobs的倔强）的标志性事件 |
| 2012年10月 | 解雇Scott Forstall | 整合内部权力，消除Jobs时代的派系冲突。Forstall被解雇后Ive获得更大设计权力 |
| 2013年 | 聘请Lisa Jackson（前EPA局长）负责环境事务 | 环保战略正式启动 |
| **2014年9月9日** | **Apple Watch发布** | Cook时代第一个全新产品类别。在Mac发布的同一场地（Flint Center）发布 |
| 2014年10月 | Apple Pay推出 | 进军金融服务 |
| **2014年10月30日** | **公开出柜**（Bloomberg Businessweek署名文章） | 成为Fortune 500首位公开出柜的在任CEO。"I'm proud to be gay, and I consider being gay among the greatest gifts God has given me." |
| 2015年4月 | Apple Watch上市（含$17,000金版） | 初始奢侈品定位 |
| 2015年6月 | Apple Music推出（基于Beats收购） | Services战略关键一步 |
| **2016年2月** | **FBI加密之争**——发布公开信拒绝破解iPhone | Cook最强力的价值观声明。"The United States government has demanded that Apple take an unprecedented step which threatens the security of our customers." |
| 2016-2017年 | Apple Watch从奢侈品转向健康定位 | Series 2加GPS，逐步放弃金版，转向运动健康 |
| 2017年 | App Store中国区下架VPN应用 | 中国妥协争议的开始 |
| 2017年6月 | MIT毕业演讲——"Be a builder" | 核心价值观公开表达 |
| 2018年 | iCloud中国数据迁移至云上贵州 | 中国合规措施，引发隐私双重标准批评 |
| **2018年8月** | **Apple成为首家市值突破1万亿美元的美国公司** | Cook时代的里程碑 |
| 2018年9月 | Apple Watch Series 4推出ECG功能（FDA批准） | Apple Watch健康战略的决定性转折 |
| 2018年5月 | Duke大学毕业演讲——"Find your fearlessness" | — |
| 2019年6月 | 斯坦福毕业演讲——"Chaos factory"批评科技行业 | 罕见地公开批评硅谷同行 |
| 2019年 | Jony Ive离开Apple | 标志着设计至上文化的衰落。Cook时代运营/Services优先 |
| 2020年7月 | 国会反垄断听证会（与Bezos、Zuckerberg、Pichai同台） | Cook表现最"安全"，几乎没给出任何可被利用的新信息 |
| **2020年6月** | **WWDC宣布Apple Silicon计划** | 开始从Intel迁移到自研芯片。"The most significant transition in the history of the Mac." |
| 2020年7月 | 宣布2030年全供应链碳中和目标 | 环保承诺最高点 |
| 2020年11月 | M1芯片发布 | Apple Silicon首代产品，性能/功耗比碾压Intel |
| 2021年 | Apple市值突破2万亿 | — |
| 2021年 | iOS 14.5推出ATT（App Tracking Transparency） | 隐私战略最具攻击性的一步，重创Meta广告业务 |
| 2021年 | Epic v. Apple判决 | 10个诉讼点赢9个，但须允许外部支付链接 |
| 2022年 | Apple Watch Ultra发布 | 极限运动定位，产品线扩展 |
| **2023年6月** | **Vision Pro在WWDC发布** | "空间计算"新品类。$3,499定价 |
| 2023年 | Apple市值突破3万亿美元 | Cook时代市值从3500亿→3万亿，增长近9倍 |
| 2024年2月 | Vision Pro上市；Project Titan（汽车项目）取消 | 一进一退：新平台赌注开始，10年汽车项目止损 |
| **2024年6月** | **Apple Intelligence发布** | AI战略正式亮相。设备端优先+隐私框架，与OpenAI合作 |
| 2024年 | 欧盟DMA生效，Apple被迫开放侧载 | App Store商业模式面临最大挑战 |

## 思想转折点总结

| 时间 | 转折 | 影响 |
|------|------|------|
| 1998年 | 放弃安全工作加入濒危的Apple | 直觉驱动的人生最大赌注 |
| 2011年 | 从二号变一号 | 从"执行者"到"决策者"的身份转变 |
| 2014年 | 公开出柜 | 从私人到公共人物的价值观表达 |
| 2016年 | FBI事件 | 将隐私从技术特性升维为道德立场和品牌核心 |
| 2016-2017年 | Apple Watch转向健康 | 学会了"从用户数据中发现产品方向"而非预设定位 |
| 2020年 | Apple Silicon | 证明垂直整合是Apple最深的护城河 |
| 2024年 | 砍掉Project Titan → All in AI | 资源重新分配的杀伐果断 |
